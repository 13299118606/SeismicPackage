CPSeis

CPSeis is the open-source version of ConocoPhillips' former seismic processing system. This system has roots back to the '60s but was completely rewritten in 1986 to operate efficiently on the Cray X-MP/14. Conoco completely rewrote the system again in 1999-2002 using the Fortran 90 programming language with C/C++ layers for I/O underneath. The new system was designed using an MPI-parallel model and works well on Linux clusters or on individual workstations.

CPSeis consists of a full set of processing modules for all aspects of seismic processing with the following notable exceptions:

  • prestack data inversion
  • prestack depth / time migration
  • multi-component tools
